# How to use this pack

1. Create or open the `trading-office` repository.
2. Copy `docs/fable-input/` into the repository.
3. Optionally place private mood screenshots into `docs/fable-input/references/`.
4. Keep raw reference screenshots out of public git unless rights are clear.
5. Give Fable the prompt from `docs/fable-input/09-fable-master-prompt.md`.
6. Ask Fable to implement only the Office Visual Builder Kit and visual example floor.
7. After visual approval, hand the resulting kit to Superpowers for the full app.
