# Moodboard Notes

Place private/local screenshots or sketches in this folder only if rights are clear or they are ignored by git.

The YouTube AI-office screenshot can be used as a private mood reference.

Do not copy it into the final product.

Useful high-level ideas from the reference:

- agents as office workers;
- desks and monitors make roles obvious;
- scene fits into one canvas;
- status labels help readability;
- the office should feel alive.

Avoid:

- copying exact desk positions;
- copying exact characters;
- copying color palette;
- copying wall/floor layout;
- too much visual noise;
- tiny text that cannot be read in demo.
