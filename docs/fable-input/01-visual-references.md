# 01 — Visual References

## Target concept

```text
AI Research Tower
```

The first visual slice is one floor:

```text
Trading Lab Research Floor
```

The visual metaphor:

- a retro pixel-art AI research office;
- agents as employees/research operators;
- a trading control-room feeling;
- visible desks, monitors, dashboards, boards, server racks;
- cozy but technical;
- readable at approximately 1280×720.

## Mood reference

A screenshot from a YouTube AI-office video may be provided locally under:

```text
docs/fable-input/references/
```

This screenshot is **mood reference only**.

Do NOT copy:

- exact layout;
- exact character design;
- exact props;
- exact palette;
- exact composition;
- exact object placement.

Extract only high-level ideas:

- agents represented as office workers;
- top-down / slightly angled office view;
- desks make roles visible;
- monitors/dashboards make the system feel alive;
- status labels make activity readable.

## Original visual direction

Style name:

```text
Retro Pixel AI Research Tower
```

Desired mood:

```text
cozy night research office + trading command center
```

Visual traits:

- dark graphite/navy floor;
- steel/dark walls;
- cyan/green terminal accents;
- amber warning accents;
- violet/gold Boss/Orchestrator accents;
- readable role labels;
- subtle monitor glow;
- not too noisy;
- no tiny unreadable text.

## Camera / composition

Preferred view:

```text
top-down or slight 3/4 top-down pixel-art office
```

Target viewport:

```text
1280×720 or scalable 16:9
```

The first floor should fit mostly on one screen. Pan/zoom can exist, but the default view should be good for demos.

## What to avoid

Avoid:

- generic RPG tavern look;
- excessive cyberpunk clutter;
- tiny detailed text;
- direct Black Mirror imitation;
- direct YouTube reference imitation;
- random mismatched asset packs;
- overly cartoonish characters;
- photorealistic UI;
- isometric complexity if it slows down implementation.

## Role-based visual identity

### Boss / Orchestrator

- central command console;
- larger desk/console;
- violet/gold accent;
- routing/control vibe;
- should visually feel like the person/system coordinating the whole floor.

### Strategy Analyst

- chart terminal;
- market notes;
- small analytics dashboard;
- calm analytical style.

### Researcher

- documents/books/idea cards;
- research desk;
- maybe small lamp and notes.

### Critic / Risk Reviewer

- red/amber review board;
- warning markers;
- checklist/review vibe.

### Builder

- dev terminal;
- code monitors;
- near server rack or technical equipment.

### Evaluator

- backtest dashboard;
- metrics display;
- chart/curve monitor.

### Performance Monitor

- bot status screen;
- heartbeat indicators;
- uptime/health vibe.

### Archive / Knowledge Base

- archive shelf/cabinet;
- glowing folders/cards;
- knowledge storage visual.
